# Spark with Mesos on Docker
---
### 1. Configuration

	$ ./config
### 2. Build
### 3. Run
 * Start the cluster

	    $ ./run start
 * Stop the cluster

	    $ ./run stop
 * Restart the cluster (=stop && start)

	    $ ./run restart
 * Remove all containers

	    $ ./run clean
 * Reset everything and start over

	    $ ./run reset
